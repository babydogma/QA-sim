# QA Simulator MVP

Статический MVP тренажёра manual QA.

## Что внутри
- `index.html`
- `style.css`
- `script.js`

## Как запустить
Просто открой `index.html` локально или залей файлы в GitHub Pages / Netlify / Vercel.

## Что уже работает
- главный экран;
- интерактивный login-сценарий;
- намеренно заложенные баги;
- форма bug report;
- примитивная оценка по ключевым словам;
- review screen;
- сохранение количества отправленных report в localStorage.

## Заложенные баги в сценарии
- кнопка Login работает при пустом password;
- email почти не валидируется;
- сообщение об ошибке слишком общее и не соответствует проблеме;
- логика состояний сделана специально криво для тренировки.

## Важно
Это не финальный продукт, а быстрый прототип, чтобы проверить идею руками.
